# Instructions

All RISC63 instructions are 16 bits wide.

## Summary

* ld, st
* addi, addui, auipc, li
* jz, jnz, aipc, jr
* slti, sltui, sgti, sgtui, srli, slli, srai, rsbi
* slt, sltu, sgt, sgtu, srl, sll, sra, rsb, and, or, xor, add, sub
* crr, crw
* jmp
* mv
* nop, sysret

More will be added very soon...

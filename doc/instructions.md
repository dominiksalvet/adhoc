# Instructions

All RISC63 instructions are 16 bits wide.

## Summary

* ld, st
* addi, addui, auipc, li
* jz, jnz, aipc, jr
* slti, sltui, sgti, sgtui, srli, slli, srai, rsubi
* slt, sltu, sgt, sgtu, srl, sll, sra, rsub, and, or, xor, add, sub
* crr, crw
* ji
* mv
* nop

More will be added very soon...
